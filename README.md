# CEMETERY MANAGEMENT SYSTEM (Abraham Bossom Memorial Cemetery)

## Getting Started

### Installation

``` bash
$ git clone https://github.com/sendowyehert01/AbrahamBossom.git

Install packages:

$ composer install
```
